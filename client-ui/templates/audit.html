Check up if vote has been counted:




* The updated encrypted tally is registered in the database everytime a voter cast a ballot. Only the voter have the hash, value which is generated right after casting the ballot by adding the values of the ciphertext values in the ballot and hash them. When the voter wants to check their ballot, they enter the hash value they got after they vote. The program will search for that hash value in the election database, get the ciphertext of the encrypted tally, perform the same hashing and compares the result with the one entered by the voter. If it's the same value, then the vote has been counted and not tamper, since we trust the service to record the vote as we casted it, but if the hash value is different, then the vote has been manipulated from the database side. {% endblock content %}
